# Barua Construction Website

A premium, responsive real estate and construction website built with Next.js 15, TypeScript, Tailwind CSS, Radix UI, and Framer Motion.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Content

Editable company and project content lives in:

- `data/settings.ts`
- `data/projects.ts`
- `data/about.ts`

The Shri Bishnu page intentionally uses clear placeholder states for details that have not yet been provided.
